# **App Name**: Al-Bashmohandes

## Core Features:

- Secure Student Authentication & Profile Management: Robust student registration and login system with detailed profile creation, capturing name, phone numbers (student & parent/guardian), and academic year. An initial admin account will be provided ready for use.
- Comprehensive Course & Video Management: Admin dashboard functionality to upload courses, set individual course prices, and integrate video content via YouTube links. Only enrolled students can access video content, ensuring content security.
- Dynamic Quiz & Exam System: Admin tools to create diverse quizzes and exams including Multiple-Choice Questions (MCQs) and Essay questions. Questions can be entered as text or uploaded as images. Essay answers can be submitted as text or PDF files, with admin review capabilities and control over when exam results are released to students.
- Code-Based Course Access Control: Admin generates unique, limited-use access codes for each course, stored securely in the admin panel. Only these codes unlock course enrollment. The admin dashboard provides a complete overview of generated codes, their usage status, and associated student accounts.
- Real-time Data Synchronization: A robust, cloud-based database ensures that all actions from the admin dashboard (e.g., new courses, student account management, progress updates) are instantly reflected across student interfaces, and student activities immediately update the admin's view.
- Student Activity & Progress Tracking: Admin dashboard module to monitor student engagement in real-time, including specific video views within courses, overall course progress, and quiz/exam submission statuses. This also includes the ability to manage and view student login details.
- AI-Powered Content Assistance Tool for Admin: An AI tool integrated within the admin dashboard to assist in content creation, capable of suggesting quiz questions or providing insights/summaries of student essay responses, thereby streamlining administrative tasks.

## Style Guidelines:

- The application will feature a dark theme. The background color will be a deep, professional black: '#1A1A1A', providing a sleek and modern foundation.
- The primary color will be a vibrant gold/yellow: '#FFD700'. This color will be used to highlight key elements, interactive components, and branding, symbolizing engineering, excellence, and energy.
- An accent color of a professional and subtle green: '#4CAF50' will be used for success indicators, alerts, or complementary design elements, providing a fresh contrast and reinforcing positive actions.
- Headlines will utilize 'Space Grotesk' (sans-serif), chosen for its contemporary and slightly technical feel, resonating with an engineering educational platform. Body text will use 'Inter' (sans-serif), valued for its exceptional legibility across various content types and screen sizes, ensuring clarity for educational materials.
- Clean, minimalist line icons will be employed throughout the platform. This style supports a modern and professional aesthetic, ensuring that all actions and content types are easily recognizable and understood by high school students. Icons will be consistent in design to maintain a cohesive visual experience.
- The platform will boast a fully responsive and dynamic layout, optimized for seamless viewing across mobile, tablet, and desktop devices. It will include a professional welcome page showcasing latest courses with an engaging design. A prominent side navigation menu will ensure intuitive access to all sections for both students and administrators. A discrete footer will display 'Made by: Mohamed Alaa'.
- Subtle and elegant animations will be incorporated for transitions between views, loading indicators, and interactive UI elements. These thoughtful animations will contribute to a smooth, dynamic, and premium user experience, enhancing the overall impression of the platform.
- The platform will support Right-to-Left (RTL) text and layout for all Arabic content, ensuring a natural and culturally appropriate reading experience for users.
