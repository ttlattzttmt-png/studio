
# منصة البشمهندس التعليمية (eplat)

هذا هو الكود المصدري لمنصة البشمهندس التعليمية، مبني باستخدام Next.js و Firebase و Genkit للذكاء الاصطناعي. المنصة مجهزة بنظام إدارة كورسات، أكواد تفعيل، وامتحانات إلكترونية ومقالية مدعومة بالذكاء الاصطناعي.

## 🚀 كيفية رفع المشروع على GitHub الخاص بك:

لرفع الملفات إلى مستودعك `https://github.com/ttlattzttmt-png/eplat` اتبع الخطوات التالية في الـ Terminal:

1. **تهيئة Git في مجلد المشروع:**
   ```bash
   git init
   ```

2. **إضافة رابط المستودع البعيد:**
   ```bash
   git remote add origin https://github.com/ttlattzttmt-png/eplat.git
   ```

3. **إضافة كافة الملفات:**
   ```bash
   git add .
   ```

4. **تسجيل التغييرات (Commit):**
   ```bash
   git commit -m "إطلاق المنصة التعليمية - البشمهندس"
   ```

5. **رفع الكود للمستودع:**
   ```bash
   git branch -M main
   git push -u origin main
   ```

## 🛠️ المميزات الرئيسية:
- **نظام الطلاب**: تسجيل حسابات، تفعيل كورسات بالأكواد، مشاهدة فيديوهات، وتأدية امتحانات.
- **نظام المسؤول (Admin)**: إنشاء كورسات، توليد أكواد دفع، بناء امتحانات، وتصحيح يدوي/آلي.
- **الذكاء الاصطناعي**: توليد أسئلة امتحانات بضغطة زر، وتحليل الإجابات المقالية للطلاب.
- **التزامن اللحظي**: كافة البيانات تتحدث فوراً لدى جميع المستخدمين باستخدام Firestore real-time.

**ملاحظة هامة:** تم تجاهل ملف `.env` و `node_modules` في الرفع لضمان أمان مفاتيحك وسرعة الرفع.
