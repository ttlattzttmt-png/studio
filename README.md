# منصة البشمهندس التعليمية (eplat)

هذا هو الكود المصدري لمنصة البشمهندس التعليمية، مبني باستخدام Next.js و Firebase و Genkit للذكاء الاصطناعي.

## خطوات رفع المشروع على GitHub الخاص بك:

لرفع الملفات إلى مستودعك `https://github.com/ttlattzttmt-png/eplat` كرر الخطوات التالية في الـ Terminal الخاص بك:

1. **تهيئة Git:**
   ```bash
   git init
   ```

2. **إضافة الملفات:**
   ```bash
   git add .
   ```

3. **تسجيل التغييرات (Commit):**
   ```bash
   git commit -m "Initial commit - Al-Bashmohandes Platform"
   ```

4. **ربط المستودع البعيد:**
   ```bash
   git remote add origin https://github.com/ttlattzttmt-png/eplat.git
   ```

5. **رفع الكود:**
   ```bash
   git branch -M main
   git push -u origin main
   ```

**ملاحظة هامة:** تم تجاهل ملف `.env` تلقائياً لحماية مفاتيح الـ API الخاصة بك من التسريب العلني.
