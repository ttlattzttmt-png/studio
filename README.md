# منصة البشمهندس التعليمية (eplat) - النسخة النهائية المستقرة

هذا هو الكود المصدري الكامل للمنصة، جاهز للرفع والتشغيل الفوري.

## 🚀 كيفية رفع المشروع على GitHub الخاص بك:

اتبع هذه الخطوات بدقة في الـ Terminal داخل مجلد المشروع:

1. **تهيئة Git:**
   ```bash
   git init
   ```

2. **إضافة رابط مستودعك:**
   ```bash
   git remote add origin https://github.com/ttlattzttmt-png/eplat.git
   ```

3. **إضافة كافة الملفات:**
   ```bash
   git add .
   ```

4. **تسجيل التغييرات:**
   ```bash
   git commit -m "الإطلاق النهائي للمنصة - نسخة مستقرة 100%"
   ```

5. **رفع الكود:**
   ```bash
   git branch -M main
   git push -u origin main
   ```
